技术准备

  1. Tor & Tor-Browser
  2. 使用虚拟机操作；用 Tails OS (更佳)
  3. 网卡MAC地址随机化
  4. 浏览器 User-Agent 伪装
  5. 文档、影像删去 metadata
  6. 使用不需要手机号注册的、且最好是去中心化的通讯软件 (e.g. Tox, Matrix)
  7. 放弃在手机上追求匿名

策略选择

  • 网上身份管理
    • 隔离：墙内平台与墙外平台
      • 不使用与实名制 ID 相同用户名、ID 和内容, etc
    • 匿名 ID 一直保持用 Tor ip 登录
    • 匿名 ID 绑定的邮箱是专用的，未关联其他 ID
  • 20个 Don't (不要去尝试的操作)

  • 更详细的匿名策略
