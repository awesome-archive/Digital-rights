6/24
  • 文件彻底删除
    • Linux: dd, wipe, securedelete, BleachBit
    • macOS: srm, rm -P 命令 (>= 10.12)
    • Windows: BleachBit
    • 替代方案：全盘加密
    • 注：对于固态硬盘（SSD）、USB盘、SD卡，所有工具都不能保证彻底删除
      • 替代方案：覆盖磁盘/分区所有空间（参见：https://gitlab.com/atgfw/Digital-rights/blob/main/T教程和笔记/2021-06-13-彻底删除-macOS-Linux.md ）

12/24
  • web 浏览
    • 最大程度防止个人信息/踪迹的泄漏
      • 使用代理(VPN)（Tor更佳）
      • 关闭 webRTC (防止泄漏真实 ip)
        • ip.voidsec.com
      • 防止DNS请求也泄漏
        • dnsleaktest.com
      • 尽量不载入 Javascript 脚本
      • No cookie, 不保存浏览历史, etc
      • 对抗浏览器指纹？--> 虚拟机+live OS
        • https://coveryourtracks.eff.org/

22/24

  • 文件、影像脱敏（去除元数据）：

    • ExifTool (未必能删去所有的元数据)
      • https://exiftool.org
    • ExifTool 在線版：https://jimpl.com
    • MAT (Metadata Anonymisation Toolkit)
      • https://mat.boum.org/
    • Document Inspector (不开源)
    • FFmpeg
    • Peepdf (可用于去除 doc/pdf 文档内的图片的元数据)
    • First Look Media's PDF Redact Tools: Flattening tool -- 媒体压平工具：如 pdf 转图片
    • Tesseract: -- 图片转文字
    • 另类技术：（古老的）翻拍翻录 -- 对图片截图/拍照，用录音笔翻录录音等
    • Signal 自带 de-metadata 功能：把图片用 Signal 发给自己，再保存

  • 其他有用的图影音编辑工具：

    • GIMP
    • Audacity
